# Demo projects hub

RubberDuck-styled control plane for optional lab demos that ship beside `demoapp/`.

**Windows:** double-click `start-hub.bat`  
**Linux / macOS:** `chmod +x start-hub.sh && ./start-hub.sh`

Open **http://127.0.0.1:5055/**

- **Welcome** — choose a demo
- **Launch** — live setup console + health diagram
- **Generate prompts** — UC-01…UC-10 prompts for the selected project (copy for Cursor / Claude)

Demos are child folders with a `setup_and_run.py` (currently: `rubberduck_pizzeria-demoapp`).
