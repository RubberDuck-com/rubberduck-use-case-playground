# UC-09: Compare Versions (Compare Versions)

## What this does

Compare two implementations or refs with a structured equivalence verdict.

## What you gain

Side-by-side divergence matrix — what changed, what stayed the same.

## How to test (2 steps)

**Step 1 — Terminal** (see the real behavior):

```bash
python scripts/run-lab.py --uc 09
```

Prints HTML vs Epub3 `globalcontext` diff live.

**Step 2 — Cursor** (paste the prompt below into chat after RubberDuck MCP is connected):

- Index once: `Index my local project at: <path-to-this-repo>`
- **Edit only:** Fill Side A / Side B (HTML vs Epub3 `prepare_writing`).
- Copy the entire **Playground prompt** block below

## What a good answer looks like

`COMPARISON_REPORT.md` with equivalence verdict.

Must include: repo name, branch/ref, **pinned commit SHA**, and **file:line** evidence.

**Focus files in this repo:** `demoapp/builders/html.py` — HTML vs Epub3 `prepare_writing`
