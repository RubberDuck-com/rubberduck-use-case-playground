# UC-10: Quick Check (Quick Check)

## What this does

One narrow yes/no question, still evidence-bound.

## What you gain

Fast answer with file:line proof; states what was not checked.

## How to test (2 steps)

**Step 1 — Terminal** (see the real behavior):

```bash
python scripts/run-lab.py --uc 10
```

Runs `render_partial` with and without input.

**Step 2 — Cursor** (paste the prompt below into chat after RubberDuck MCP is connected):

- Index once: `Index my local project at: <path-to-this-repo>`
- **Edit only:** Fill `Question` about `render_partial` in the HTML builder.
- Copy the entire **Playground prompt** block below

## What a good answer looks like

`QUICK_CHECK.md` with yes/no and evidence table.

Must include: repo name, branch/ref, **pinned commit SHA**, and **file:line** evidence.

**Focus files in this repo:** `demoapp/builders/html.py` — `render_partial`
